# Scanner de Picks

App estática (HTML/CSS/JS, sin dependencias) que ayuda a analizar apuestas deportivas combinando la cuota con factores de contexto (forma reciente, h2h, bajas, estilo de juego, etc.) para generar un índice de confianza con razonamiento explicado.

⚠️ No es una garantía de resultados. Es una herramienta de apoyo al análisis.

## Uso

Abrir `index.html` en cualquier navegador. No requiere build ni servidor.

### Publicar con GitHub Pages

1. Crear un repo en GitHub y subir este contenido.
2. Ir a **Settings → Pages**.
3. En "Source" elegir la rama `main` y carpeta `/ (root)`.
4. Guardar — la app quedará disponible en `https://TU-USUARIO.github.io/NOMBRE-REPO/`.
